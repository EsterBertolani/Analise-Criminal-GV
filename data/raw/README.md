# Dados brutos

Coloque aqui o arquivo oficial de ocorrências usado no projeto:

```text
data/raw/MICRODADOS_OCORRENCIAS.csv
```

O repositório vem com um arquivo sintético opcional (`MICRODADOS_OCORRENCIAS_DEMO.csv`) apenas para testar o pipeline e a interface sem depender do microdado oficial.

Para gerar novamente os dados de demonstração:

```bash
python scripts/generate_demo_data.py
```
